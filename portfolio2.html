PhoneCheckCloud

Project information

  • Category: Web design & Development
  • Company/Client: Upgenics International
  • Project start date: July, 2019
  • Project end date: December 2019
  • Project URL: https://www.phonecheck.com
  • Tech stack: PHP, Codeigniter (HMVC), Laravel, Mysql, JavaScript, JQuery

As a Full Stack Developer at Phonecheck, a leading diagnostic platform for pre-owned mobile devices, I played a central role in the comprehensive enhancement of the Phonecheck Cloud web application. In this capacity, my responsibilities extended to the modernization of legacy portals, development of cross-platform APIs, and addressing foundational issues within the company's cloud infrastructure.

  • Revamped Phonecheck Cloud

    - Initiated a substantial overhaul of Phonecheck Cloud, a critical web application managing data synchronization from the Phonecheck desktop application.
    - This diagnostic platform conducts comprehensive analyses on pre-owned mobile devices, offering detailed insights into hardware and software components.

  • Modernized Legacy Portal:

    - Skillfully maintained and modernized the legacy portal of Phonecheck Cloud, originally built on CodeIgniter 2 with HMVC.
    - This involved eliminating outdated code and implementing updates to significantly enhance system performance.

  • API Development for Cross-Platform Communication:

    - Played a pivotal role in developing APIs catering to both desktop and mobile applications, ensuring seamless communication and integration across platforms.

  • Cloud Infrastructure Enhancement:

    - Addressed foundational issues in the company's cloud infrastructure, yielding tangible benefits for esteemed clients like Brightstar and Gazelle in the mobile diagnostics sector which contributed to operational optimization and achieving client satisfaction.

In summary, In my capacity at Phonecheck, I led the strategic revamp of the Phonecheck Cloud, modernized legacy portals, and developed cross-platform APIs. These efforts not only improved system performance but also addressed critical issues in the cloud infrastructure, providing tangible benefits for high-profile clients and enhancing overall client satisfaction.
